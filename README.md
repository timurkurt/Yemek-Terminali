# Yemek Terminali

Merhaba biz İstanbul Gelişim Üniversitesi Bilgisayar Teknolojileri Öğrencileri Timur Kurt ve Recep Demir, restoranlarda yaşadığımız sorunlarla ilgili bu projeyi çıkattık. Bu proje, restoranlar ve küçük işletmeler için geliştirilmiş, dokunmatik ekran uyumlu bir Sipariş Terminal uygulamasıdır. Kullanıcıların menüleri incelemesine, ekstralarını seçmesine ve siparişlerini tamamlayıp adisyon çıktılarını almasına olanak tanır.

## Öne Çıkan Özellikler

**Dinamik Video Arka Planı:** Uygulama bekleme modundayken arkada iştah açıcı videolar oynatır

**Gerçek Zamanlı Sepet Hesaplama:** Seçilen her ürün ve ekstra için anlık fiyat güncellemesi yapar.

**Otomatik Adisyon Sistemi:** Sipariş tamamlandığında hem mutfağa gönderir, hemde projenin içine entegre edilmiş yazıcı seçmeye gerek kalmadan varsayılan yazıcıdan replikasını çıkarmakta.

**Kapatılamaz Arayüz:** Tkinter kullanılarak tasarlanmış, tam ekran modunda çalışır bu sayede programdan çıkış yapmak zorlaştırılmıştır

## Kullanılan Kütüphaneler
**Python 3.x**

**Tkinter:**  Grafik arayüz tasarımı.

**OpenCV (cv2):** Video işleme ve oynatma.

**Pillow (PIL):** Görsel işleme ve buton tasarımları.

**PyWin32:** Windows yazıcı (printer) entegrasyonu.

# Food Terminal

Hello, we, Istanbul Gelisim University Computer Technologies Students Timur Kurt and Recep Demir, have come up with this project about the problems we experience in restaurants. This project is a touch screen compatible Order Terminal application developed for restaurants and small businesses. It allows users to review menus, select extras, and complete their orders and print out their tickets.

## Highlights

**Dynamic Video Background:** Plays appetizing videos in the background when the app is in standby mode 

**Real-Time Cart Calculation:** Provides instant price updates for each selected product and extra. 

**Automatic Billing System:** When the order is completed, it both sends it to the kitchen and outputs the replica from the default printer integrated into the project. 

**Unpluggable Interface:** Designed using Tkinter, it operates in full-screen mode, making it difficult to log out of the program

## Libraries Used 
**Python 3.x** 

**Tkinter:** Graphic interface design. 

**OpenCV (cv2):** Video processing and playback. 

**Pillow (PIL):** Visual processing and button designs. 

**PyWin32:** Windows printer integration.
